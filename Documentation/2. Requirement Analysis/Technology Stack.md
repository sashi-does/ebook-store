# Technology Stack Selection

## Backend Services
- **Runtime Environment**: Node.js (v16+)
- **Server Framework**: Express.js
- **Database**: MongoDB Atlas (Cloud)
- **Object Data Modeling (ODM)**: Mongoose (v8+)
- **Security**: JSON Web Tokens (`jsonwebtoken`) & `bcryptjs`
- **Cross-Origin Resources**: `cors` middleware

## Frontend Application
- **Build Tool**: Vite (extremely fast hot module reloading)
- **Library**: React.js (v18+)
- **Routing**: React Router DOM (v6)
- **Network Interface**: Axios
- **Styling**: Vanilla CSS featuring variables, CSS transitions, and glassmorphic filters.

## Developer Utilities
- **Process Manager**: Concurrently (run client and server concurrently)
- **Auto-reloading**: Nodemon (restart backend on code changes)
