## Key Demonstration Flows

1. **User Authentication & Role Switch:**
   - Register a new user.
   - Login as a customer.
   - Show dashboard restrictions for customers.
   - Log in as administrator (`dsp_admin`) to unlock the Admin Dashboard.

2. **Catalog & Search Operations:**
   - Search for books by title/author/genre.
   - Filter book results.
   - Click on a book to view detailed information, inventory, and reviews.

3. **Shopping Cart & Checkout:**
   - Add items to the cart.
   - Modify quantities in the cart page.
   - Proceed to checkout and submit shipping details.

4. **Order Status Lifecycle:**
   - View order confirmation and check order tracking page.
   - Access admin panel to change order status (e.g., from `Processing` to `Shipped`).
   - Verify status updates on user dashboard.

5. **Customer Reviews:**
   - Submit rating and comment for a purchased book.
   - Attempt duplicate review submission (to verify unique constraint enforcement).
