# Empathy Map Canvas - Book Haven User Profiles

## The Avid Reader (Customer Profile)

```
+-----------------------------------------------------+-----------------------------------------------------+
| SAYING                                              | THINKING                                            |
| - "I want to quickly find new Sci-Fi books."        | - "I hope the checkout is fast and doesn't crash."  |
| - "Is this book highly rated by others?"            | - "Will these reviews be reliable?"                 |
+-----------------------------------------------------+-----------------------------------------------------+
| DOING                                               | FEELING                                             |
| - Filters catalog by rating and price range.        | - Excited to discover fresh books.                  |
| - Creates account to save order details.            | - Pleased by the smooth visual flow of the UI.      |
+-----------------------------------------------------+-----------------------------------------------------+
```

## The Store Owner (Administrator Profile)

```
+-----------------------------------------------------+-----------------------------------------------------+
| SAYING                                              | THINKING                                            |
| - "I need to see today's orders and revenue."       | - "Updating the catalog must be simple."            |
| - "How can I update this book's price quickly?"     | - "I need to monitor low-stock levels easily."      |
+-----------------------------------------------------+-----------------------------------------------------+
| DOING                                               | FEELING                                             |
| - Checks dashboard KPIs daily.                      | - Confident managing inventory logs.                |
| - Adjusts stock levels on catalog tables.           | - Reassured by clean, simple dashboard metrics.     |
+-----------------------------------------------------+-----------------------------------------------------+
```
